## 💧 Overview  
A **next-gen UI concept** where the interface behaves like liquid — smooth, reactive, and alive.  
This project focuses on **fluid animations, ripple effects, and glassmorphism design** to create a futuristic experience.

---

## 🎨 Visual Experience  
- 🌊 Floating liquid blobs  
- 🌀 Ripple interaction on cursor movement  
- 💎 Glassmorphism UI card  
- 🌈 Smooth wave-like button effects  

---

## ⚡ Features  
- 🌊 Ripple Engine → Mouse movement creates water-like ripples  
- 🔵 Liquid Background → Animated glowing blobs simulate fluid motion  
- 💎 Glass UI → Frosted glass card with blur effects  
- 🌀 Smooth Animations → Soft transitions and wave interactions

## 🧪 Tech Stack  
```diff
+ HTML5        → Structure
+ CSS3         → Liquid animations + glassmorphism
+ JavaScript   → Ripple physics & interaction

🧠 How It Works
Canvas API → Generates ripple waves
CSS Keyframes → Drives blob animations
Backdrop Filter → Creates glassmorphism
Event Listeners → Tracks cursor movement
